# KIDS-WORLD

Premium newborn essentials, baby products, and thoughtfully curated gifts for every parent.

## Overview

KIDS-WORLD is a modern, responsive baby products storefront built with React, TypeScript, Vite, and Tailwind CSS. The project includes a mobile-first layout, featured products, category browsing, contact details, and production-ready build output for deployment.

## Tech Stack

- React 19
- TypeScript 5
- Vite 6
- Tailwind CSS 4
- Motion
- Lucide React

## Getting Started

1. Clone the repository.
2. Change into the project directory:

```bash
cd KIDS-WORLD
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## Available Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
